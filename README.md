## 🚀 Descripción

Aplicación web desarrollada con **JavaScript** que implementa un editor Markdown interactivo, permitiendo la creación, edición y persistencia de contenido directamente en el navegador.

El proyecto está enfocado en consolidar fundamentos clave del lenguaje, incluyendo manipulación del DOM, estructuras de datos y gestión de estado mediante almacenamiento local.

## 🎯 Objetivos

- Implementar un editor Markdown funcional desde cero  
- Aplicar manipulación dinámica del DOM  
- Gestionar persistencia de datos con `localStorage`  
- Estructurar lógica de aplicación basada en JavaScript puro  
- Construir una base sólida para aplicaciones más complejas  

## 🛠️ Tecnologías Utilizadas

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Web APIs (DOM, Local Storage)  

## 📌 Características

- Editor Markdown en tiempo real  
- Persistencia automática en el navegador  
- Renderizado dinámico del contenido  
- Interfaz simple y enfocada en productividad  
- Arquitectura basada en fundamentos (sin frameworks)  

## 🧠 Conceptos Aplicados

- Manipulación del DOM  
- Gestión de eventos  
- Estructuras de datos en JavaScript  
- Separación de responsabilidades  
- Persistencia en cliente (localStorage)  

## 🔮 Evolución del Proyecto

Este proyecto está diseñado como una base técnica. Próximas iteraciones podrían incluir:

- Sincronización en la nube  
- Sistema de autenticación  
- Exportación de documentos  
- Uso de frameworks modernos (React, Vue)  
- Soporte para temas y personalización  

## 📄 Uso

Este proyecto puede servir como referencia para:

- Comprender cómo construir un editor desde cero  
- Practicar fundamentos sólidos de JavaScript  
- Implementar almacenamiento local en aplicaciones web
